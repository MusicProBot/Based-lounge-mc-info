How to Install Mods

Here you will find all the details
about how to install the mods on your game and SMP backup mods

Follow the steps

1: Make sure you have the minecraft launcher installed
2: Install Forge or Fabric
3: Open the file you downloaded and press to install the client
4: After that open Minecraft and you should see forge 1.21.4 automatically be selected
5: Go to Windows search and type "run" and inside the text type "%appdata%" and then press OK
6: At the top you will see a folder called ".minecraft", open that folder and inside you should see a folder called "Mods"
7: If you don't see a "Mods" folder just create a new folder and name it "mods", and make sure the folder does not start with a capital "M"
8: After that open the "All working Mods.zip" and copy everything inside the folder to the "Mods" folder in your Minecraft game
9: You can now launch Minecraft and join the server.

Additional info for smoother gameplay

1: After installing the mods, go to the Minecraft launcher
2: Press at the top that says "Installation"
3: Then you will see "Fabric-Loader OR Forge" (depends on what you installed)
4: Hover over the desired installation and at the very right you will see 3 dots
5: Press on those 3 dots and click "Edit"
6: After pressing "Edit" at the bottom you will see "More Options"
7: Press on "More Options" and if you scroll down you will see "JVM ARGUMENTS"
8: In the text box at the very left you will see "-Xmx2G"
9: Change that number to 4 or 8 depending on your computer RAM
10: After that just press save and you're ready to go ✅
"What this basically does is use more RAM and has more headroom for mods to work smoothly and the game itself."

How to install SMP Backup Mods

1: Install Fabric or Forge (you can find which mod loader each world uses in the description of the world).
2: Then download the mod folder that was included with the SMP world (it's in the description).
3: Extract the mods folder and drag all the mods into your .minecraft/mods folder.
"To find your .minecraft/mods, simply press the search on Windows and type "%appdata%".
4: After that, go to the Minecraft launcher and select Fabric or Forge (usually, it selects automatically when you install the mod loader).
5: That's it! Just press Play and enjoy the worlds ✅

How to install Backup SMP Worlds

1: Download the world of your choice.
2: After Downloading extract the folder (or you can simply open it directly).
3: after extracting drag the extracted folder in your .minecraft/saves folder.
"To find your .minecraft/saves, simply press the search on Windows and type "%appdata%".
4: After that, your ready to go and enjoy the map ✅
