How to Install Mods

Here you will find all the details
about how to install the mods on your game and SMP backup mods

Follow the steps

1: Make sure you have the minecraft launcher install
2: install Forge or fabric
3: open the file you downloaded and press to install the client
4: after that open minecraft and you should see forge 1.21.4 automatically be selected
5: go to windows search and type "run" and inside the text type "%appdata%" and then press ok
6: at the top you will a folder called ".Minecraft", open that folder and inside you should see a folder called "Mods"
7: If you don't see a "Mods" folder just create a new folder and name it "mods", and make sure the folder does not start with a capital "M"
8: after that open the "All working Mods.zip" and copy everything that is inside the folder to the "Mods" folder in your minecraft game
9: and you can now launch minecraft and join the server

Aditional info for smoother gameplay

1: After installing the mods go to the minecraft launcher
2: press at the top that says "Installation"
3: then you will see "Fabric-Loader OR forge" (depends on what you installed)
4: hover over the desire installation and at the very right you will see 3 dots
5: press on those 3 dots and click "Edit"
6: After pressing "Edit" at the bottom you will see "More Options"
7: Press on "More Options" and if you scroll down you will see "JVM ARGUMENTS"
8: In the text box at the very left you will see "-Xmx2G"
9: Change that number to 4 or 8 depenting on your computer ram
10: After that just press save and your ready to go ✅
"What this basically does it uses more ram and has more headroom for mods to work smoothly and the game itself"

How to install SMP Backup Mods

1: Install Fabric or Forge, (you can find what mod loader each world uses on the discription of the world)
2: Then download the mod folder that was included on the smp world (it's on the discription)
3: Extract the mods folder and drag all the mods in your .minecraft/mods folder
"To find your ".minecraft/mods" simply press the search on windows and type "%appdata%"
4: After that go on the minecraft launcher and select Fabric or Forge (usually it selects automatically when you install the mod loader)
5: And that it just press Play and enjoy the worlds✅
